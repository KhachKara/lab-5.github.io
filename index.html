Прогнозирование Поведения Потребителя

Цель проекта создать приложение для прогнозирования поведения потребителя. В качестве исходных данных в приложение загружаются статистические данные нескольких прошедших лет, и приложение выдает прогноз на дальнейшие действия потребителя на ближайший период времени.
